# 活动策划技能

## 核心能力

### 1. 节日活动

**全年活动日历**：

| 月份 | 节日 | 活动类型 | 建议力度 |
|------|------|----------|----------|
| 1月 | 元旦/春节 | 团圆饭、优惠 | 中 |
| 2月 | 情人节 | 双人套餐、浪漫 | 高 |
| 3月 | 女神节 | 女性专属 | 中 |
| 4月 | 踏青 | 户外套餐 | 低 |
| 5月 | 劳动节 | 致敬劳动 | 中 |
| 6月 | 618/端午 | 线上大促 | 高 |
| 7-8月 | 暑期 | 清凉主题 | 中 |
| 9月 | 教师节 | 老师专属 | 中 |
| 10月 | 国庆 | 黄金周 | 高 |
| 11月 | 双11 | 预售 | 高 |
| 12月 | 圣诞/元旦 | 跨年派对 | 高 |

**春节活动模板**：
```
【主题】：XXX
【时间】：除夕-初七
【目标】：销售XX万，新客XX人

【活动内容】：
1. 预热期：朋友圈预热
2. 爆发期：套餐优惠
3. 返场期：会员福利

【预算】：XX元
【预期ROI】：1:X
```

### 2. 主题活动

**会员日活动**：
```
【主题】：每月18日会员日
【活动】：
- 会员专属折扣：8折
- 双倍积分
- 限定礼品

【宣传】：
- 提前3天群通知
- 短信推送
- 门口易拉宝
```

**新品发布会**：
```
【流程】：
1. 预热：倒计时3天
2. 现场：试吃+介绍
3. 福利：限量免费尝
4. 传播：拍照打卡

【物料】：
- 展板
- 试吃品
- 拍照道具
- 伴手礼
```

### 3. 促销方案

**折扣设计**：

| 类型 | 公式 | 适用 |
|------|------|------|
| 全场折扣 | 8折 | 清库存 |
| 单品折扣 | 招牌7折 | 推爆品 |
| 限时折扣 | 下午2-5点7折 | 拉低谷 |
| 阶梯折扣 | 满100-10,满200-30 | 提客单 |

**买赠设计**：

| 类型 | 示例 | 目的 |
|------|------|------|
| 买一送一 | 买焗饭送饮品 | 推新品 |
| 满赠 | 满80送小菜 | 提客单 |
| 加价购 | +1元换饮料 | 清库存 |

**抽奖设计**：

| 奖项 | 奖品 | 成本 |
|------|------|------|
| 一等奖 | 免单 | 高 |
| 二等奖 | 50元券 | 中 |
| 三等奖 | 饮品 | 低 |
| 参与奖 | 积分 | 低 |

### 4. 活动执行

**执行清单**：

| 时间 | 事项 | 负责人 |
|------|------|--------|
| T-7天 | 方案确定 | 店长 |
| T-5天 | 物料制作 | 店长 |
| T-3天 | 宣传启动 | 店长 |
| T-1天 | 物料到位 | 员工 |
| T当天 | 执行 | 全体 |
| T+1天 | 复盘 | 店长 |

**物料清单**：

| 类型 | 物料 |
|------|------|
| 宣传 | 朋友圈海报、易拉宝、传单 |
| 氛围 | 气球、灯笼、KT板 |
| 物料 | 优惠券、礼品、抽奖箱 |
| 食品 | 活动套餐食材 |
