# 销售营销技能

## 核心能力

### 1. 销售技巧

**销售流程**：
```
1. 迎宾 → 打招呼，热情微笑
2. 了解 → 问几位、第一次来吗
3. 推荐 → 根据需求推荐菜品
4. 促单 → 强调价值，推动下单
5. 异议处理 → 价格贵？考虑下？
6. 成交 → 确认订单，感谢选择
7. 送客 → 邀请下次再来
```

**话术模板**：

| 场景 | 话术 |
|------|------|
| 迎宾 | "欢迎光临，请问几位？" |
| 第一次来 | "我们是做XX的，招牌是XX" |
| 推荐 | "今天想吃点清淡还是重口的？" |
| 促单 | "这道是我们主推的，很多回头客都在点" |
| 客单价提升 | "加个饮品只要+5元，很划算" |
| 催单 | "您先点着，我帮您催厨房" |
| 送客 | "感谢光临，欢迎下次再来~" |

**异议处理**：
```
客户：太贵了
→ "我们的分量是别家1.5倍，您试试看"

客户：考虑一下
→ "好的，您先看菜单，有需要随时叫我"

客户：不要推销
→ "抱歉打扰，您先看，有需要叫我"
```

### 3. 外卖运营

**平台活动**：

| 活动 | 作用 | 建议 |
|------|------|------|
| 天天神券 | 拉新 | 券额大一点 |
| 满减活动 | 提升客单价 | 设置阶梯满减 |
| 会员红包 | 复购 | 复购高必开 |
| 限时折扣 | 清库存 | 滞销品 |
| 新客立减 | 拉新 | 首单转化 |

**爆品打造**：
```
1. 选品：受众广、毛利高、复购强
2. 定价：低价引流款（不赚钱但吸引人）
3. 主图：高清食欲感
4. 评价：引导好评
5. 排名：销量+评分+转化率
```

**排名权重**：
- 销量 30%
- 评分 25%
- 转化率 20%
- 复购率 15%
- 活动力度 10%

### 4. 私域流量

**微信群运营**：
```
每日内容：
- 早：天气预报+早餐推荐
- 中：新品/活动推送
- 晚：好评截图+次日预告

互动：
- 红包雨
- 抽奖
- 接龙
```

**企微运营**：
- 1v1私信：节日祝福、生日优惠
- 朋友圈：日常分享
- 社群：活动通知

**会员体系**：
```
等级：
- 普通会员：注册
- 银卡：消费满500
- 金卡：消费满1500
- 黑卡：消费满3000

权益：
- 积分兑换
- 专属折扣
- 优先预订
- 生日礼遇
```
