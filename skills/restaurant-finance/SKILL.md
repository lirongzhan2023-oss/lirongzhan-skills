# 餐饮财务技能

## 核心能力

### 1. 成本核算

**餐饮成本构成**：

| 项目 | 占比参考 | 说明 |
|------|----------|------|
| 食材成本 | 28-35% | 原料+调料 |
| 人工成本 | 25-30% | 工资+社保 |
| 租金成本 | 8-15% | 房租+物业 |
| 水电能耗 | 3-5% | 电费+水费+燃气 |
| 耗材包装 | 3-5% | 包装+一次性用品 |
| 其他 | 5-8% | 营销+维修+折旧 |

**综合成本率**：一般控制在 **65-70%** 以内

---

### 2. 毛利计算

**公式**：
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
```

**单品毛利**：
```
单品毛利 = 售价 - 成本
单品毛利率 = (售价 - 成本) / 售价 × 100%
```

**示例**：
> 芝士焗饭售价：32元
> 食材成本：大米2元 + 芝士5元 + 配料3元 + 包装1元 = 11元
> 毛利 = 32 - 11 = 21元
> 毛利率 = 21/32 = 65.6%

**合格线**：
- 快餐简餐：55-65%
- 正餐：60-70%
- 火锅烧烤：65-75%

---

### 3. 定价策略

**成本加成法**：
```
定价 = 成本 / 目标毛利率
```

**参考定价公式**：
```
售价 = 食材成本 × (3~3.5)  // 快餐
售价 = 食材成本 × (3.5~4)  // 正餐
```

**市场比较法**：
- 参考竞品价格
- 考虑品牌溢价
- 预留活动空间

---

### 4. 盈亏平衡点

**公式**：
```
盈亏平衡点（收入）= 固定成本 / 毛利率
```

**固定成本**：
- 租金
- 人工（固定部分）
- 水电（固定部分）

**示例**：
> 月租金：15000元
> 固定人工：20000元
> 其他固定：5000元
> 固定成本合计：40000元
> 毛利率：65%
> 盈亏平衡点 = 40000 / 65% = 61538元/月

即：**月营业额≥6.15万** 才能保本

---

### 5. 投资回报测算

**单店投资模型**：

| 投资项 | 金额 |
|--------|------|
| 装修 | XX万 |
| 设备 | XX万 |
| 首月房租 | XX万 |
| 首月人工 | XX万 |
| 首月原材料 | XX万 |
| 其他 | XX万 |
| **合计** | **XX万** |

**回本周期**：
```
回本周期 = 总投资 / 月净利润
```

**参考回本周期**：
- 快餐：6-12个月
- 正餐：12-18个月
- 轻食茶饮：6-9个月

---

### 6. 财务日报/周报

**日报核心指标**：

| 指标 | 计算 | 意义 |
|------|------|------|
| 营业额 | 当天收入 | 销售情况 |
| 成本率 | 当天成本/收入 | 成本控制 |
| 客单价 | 收入/单量 | 消费水平 |
| 单量 | 订单数 | 流量情况 |
| 毛利率 | (收入-成本)/收入 | 盈利能力 |

**周报重点**：
- 环比变化
- 分类销售（堂食vs外卖）
- 爆品销售占比
- 成本异常报警
