# 社交媒体运营技能

## 核心能力

### 1. 抖音内容

**爆款结构**：

```
3秒完播率：开头要抓人
↓
15秒内容：核心价值
↓
结尾引导：点赞/关注/到店
```

**餐饮抖音类型**：

| 类型 | 时长 | 拍摄要点 |
|------|------|----------|
| 菜品展示 | 15-30秒 | 高清+食欲感 |
| 后厨日常 | 30-60秒 | 真实+有趣 |
| 探店打卡 | 1-3分钟 | 颜值+体验 |
| 剧情反转 | 1分钟 | 有趣+反转 |

**拍摄技巧**：
```
1. 光线：自然光最好
2. 角度：俯拍45度
3. 稳定：手要稳或用支架
4. 声音：清晰无噪音
5. 剪辑：节奏要快
```

**剪辑要点**：
```
1. 开头3秒必须有亮点
2. 每5秒换一个镜头
3. 背景音乐要带感
4. 字幕要大要清晰
5. 结尾引导要自然
```

### 2. 小红书运营

**笔记结构**：

```
标题：关键词+悬念/数字/情绪
封面：高清+文字+关键词
正文：痛点+方案+体验+推荐
结尾：互动引导+地点信息
```

**小红书爆款标题**：

| 类型 | 示例 |
|------|------|
| 数字型 | 3年反复去吃！这家店... |
| 情绪型 | 绝绝子！XXX也太好吃了吧 |
| 问答型 | XXX哪家强？本地人推荐... |
| 攻略型 | 保姆级XXX攻略｜少走弯路 |

**关键词布局**：

```
标题：核心词+长尾词
正文：自然分布关键词
标签：3-5个相关标签
位置：门店地址
```

**提升排名技巧**：

```
1. 关键词：标题+正文
2. 图片：高清+标签
3. 互动：评论回复
4. 收藏：内容有价值
5. 更新：保持活跃
```

### 3. 大众点评

**店铺优化**：

| 维度 | 优化点 |
|------|--------|
| 店铺名 | 品牌名+品类 |
| 头图 | 门店照/菜品照 |
| 简介 | 特色+定位 |
| 标签 | 必点+特色 |

**好评获取**：

```
1. 服务中：口头引导
2. 用餐后：二维码引导
3. 离店后：短信感谢+好评邀请
4. 好评后：回复感谢
```

**差评处理**：

```
1. 30分钟内回复
2. 态度真诚道歉
3. 提出解决方案
4. 私下联系补救
5. 定期检查回复
```

**流量获取**：

| 方式 | 作用 |
|------|------|
| 霸王餐 | 曝光+评价 |
| 推广通 | 排名提升 |
| 达人探店 | 种草 |
| 粉丝福利 | 转化 |

### 4. 视频号/快手

**视频号特点**：
- 私域流量强（朋友圈+微信群）
- 中老年用户多
- 适合本地生活

**直播话术**：

```
开场：欢迎XXX来到直播间！
产品：今天给大家介绍XXX
福利：限时优惠！仅限XXX
引导：点击下方链接！
结尾：感谢XXX，下播再会！
```

**短视频脚本**：

```
【脚本模板】

开场（3秒）：
"XXX也太好吃了吧！"

内容（30秒）：
"今天去XXX..."
"味道XXX..."
"价格XXX..."

结尾（3秒）：
"想吃的评论区告诉我！"
```
