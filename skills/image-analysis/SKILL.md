# 图像分析技能

## 核心能力

### 1. 截图数据提取

**分析流程**：
```
1. 仔细查看全图（不遗漏任何角落）
2. 识别数据类型（销售、流量、评分等）
3. 提取具体数值（单位要准确）
4. 检查数据逻辑（总和、占比）
5. 标注数据来源（平台、时间）
```

**关键检查项**：
- [ ] 店名/日期
- [ ] 核心指标（单量、收入、访客）
- [ ] 转化率、复购率、评分
- [ ] 小字/备注信息
- [ ] 趋势箭头（上升/下降）

---

### 2. 数据校验

**必须验证**：
- 数据一致性（各数据能互相印证吗？）
- 合理性（数值在正常范围吗？）
- 完整性（有没有遗漏的指标？）

**计算验证**：
- 客单价 = 收入 / 单量
- 转化率 = 订单 / 访客 × 100%
- 验证结果是否匹配截图

---

### 3. 结构化输出

**输出格式**：

```
【数据来源】

- 平台：XXX
- 时间：XXX

【核心数据】
| 指标 | 数值 | 单位 |
|------|------|------|
| 单量 | XXX | 单 |
| 收入 | XXX | 元 |
| 访客 | XXX | 人 |

【数据验证】
- 客单价 = 收入/单量 = XX元 → ✅验证通过/❌异常

【发现】
- 关键发现1
- 关键发现2

【建议方向】
- 方向1
- 方向2
```

---

### 4. 多图对比分析

**对比维度**：
- 环比（本周 vs 上周）
- 同比（本月 vs 上月）
- 跨店对比（店1 vs 店2）
- 跨平台对比（美团 vs 饿了么）

**对比输出**：

```
【对比项目】：XXX vs XXX

【指标变化】
| 指标 | 时期1 | 时期2 | 变化 |
|------|-------|-------|------|
| 单量 | 100 | 120 | +20% |
| 收入 | 3000 | 3600 | +20% |

【趋势判断】：上升/下降/持平

【原因分析】：
1. XXX
2. XXX
```

---

### 5. 餐饮外卖专业指标库

**必须关注的指标**：

| 指标 | 合格线 | 优秀 | 含义 |
|------|--------|------|------|
| 转化率 | 3% | 5-8% | 访客→下单 |
| 复购率 | 15% | 25-35% | 回头客占比 |
| 评分 | 4.5 | 4.8+ | 客户满意度 |
| 客单价 | 25元 | 35元 | 每单收入 |
| 复购频次 | 2次/月 | 4次/月 | 消费频次 |

**问题预警**：

| 指标 | 红灯 | 原因 |
|------|------|------|
| 转化率低 | <2% | 主图/价格/菜单 |
| 复购率低 | <10% | 口味/服务 |
| 评分降 | <4.5 | 差评增加 |
| 单量跌 | 环比-20% | 竞品/活动 |

## 读图检查清单

每次读取图片时必须检查：

```
□ 标题（店名、平台、时间）
□ 核心指标（单量、收入、访客）
□ 百分比指标（转化、复购、评分）
□ 趋势箭头或标记
□ 小字/备注/注释
□ 图例/分类标签
□ 数据单位（元/单/人/%）
□ 页面其他信息
```

---

## 输出规范

**格式原则**：
1. 数据必须准确提取（不猜测）
2. 带单位的数值更清晰
3. 与行业基准对比
4. 发现问题要标注

**质量标准**：
- ✅ 数值准确提取
- ✅ 单位标注清晰
- ✅ 逻辑验证通过
- ✅ 建议方向明确
