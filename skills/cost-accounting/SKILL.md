# 餐饮成本核算技能

## 1. 成本结构

### 餐饮成本分类

| 类别 | 明细 | 占比参考 |
|------|------|----------|
| 物料成本 | 食材、包材、调料 | 28-35% |
| 人力成本 | 工资、社保、福利 | 25-30% |
| 租金成本 | 房租、物业 | 8-15% |
| 运营成本 | 水电、燃气、维修 | 5-8% |
| 营销成本 | 推广、平台抽成 | 3-5% |
| 其他成本 | 设备折旧、贷款利息 | 3-5% |

### 计算公式

```
总成本 = 物料 + 人力 + 租金 + 运营 + 营销 + 其他
毛利率 = (营收 - 物料成本) / 营收 × 100%
净利率 = 净利润 / 营收 × 100%
```

---

## 2. 毛利率计算

### 单品毛利率

```
单品毛利 = 售价 - 单品成本
单品毛利率 = (售价 - 单品成本) / 售价 × 100%

示例：
- 芝士焗饭售价：32元
- 大米：2元 + 芝士：5元 + 配料：3元 + 包装：1元 = 11元
- 单品毛利：32 - 11 = 21元
- 毛利率：21 / 32 = 65.6%
```

### 品类毛利率

```
品类毛利率 = (品类营收 - 品类成本) / 品类营收 × 100%
```

### 整体毛利率

```
整体毛利率 = (总营收 - 总物料成本) / 总营收 × 100%
```

## 3. 成本预警

### 预警指标

| 指标 | 正常 | 预警 | 危险 |
|------|------|------|------|
| 物料成本占比 | 30-35% | 35-40% | >40% |
| 人力成本占比 | 25-28% | 28-32% | >32% |
| 租金成本占比 | 8-12% | 12-15% | >15% |
| 综合毛利率 | >60% | 50-60% | <50% |

### 预警机制

```
每日检查：
- 物料成本占比 > 38% → 预警
- 某品类毛利率 < 50% → 预警
- 成本环比增长 > 10% → 预警
```

## 5. 成本异常检测

### 常见异常

| 异常 | 表现 | 原因 |
|------|------|------|
| 物料超标 | 占比突然>40% | 浪费/采购涨价/偷窃 |
| 人效下降 | 人力成本↑但营收↓ | 人员冗余/效率低 |
| 损耗增加 | 报废金额↑ | 库存管理问题 |
| 采购异常 | 采购价↑但用量↓ | 供应商涨价 |

### 检测代码

```javascript
// 物料成本异常检测
if (物料占比 > 40) {
  预警('物料成本超标：' + 物料占比 + '%');
}

// 毛利率异常
if (单品毛利率 < 40) {
  预警('单品毛利过低：' + 商品名);
}

// 成本环比
if (本月成本 / 上月成本 > 1.1) {
  预警('成本增长10%以上');
}
```

## 6. 成本优化

### 优化方向

| 方向 | 动作 | 预期效果 |
|------|------|----------|
| 采购优化 | 比价/批量采购 | -5%成本 |
| 库存管理 | 先进先出 | -3%损耗 |
| 人效提升 | 排班优化 | -2%人力 |
| 菜单优化 | 主推高毛利 | +3%毛利 |
| 能源节约 | 水电管控 | -2%运营 |

### 具体方案

```
【物料降本】
1. 供应商比价 → -5%
2. 批量采购 → -3%
3. 替代食材 → -2%
合计：-10%

【人效提升】
1. 排班优化 → -3%
2. 绩效激励 → -2%
合计：-5%
```
