# 餐饮视觉设计技能

## 1. 菜单设计

### 电子菜单结构

| 板块 | 内容 |
|------|------|
| 招牌推荐 | 3-5款爆品 |
| 主食 | 焗饭、意面等 |
| 小吃 | 薯条、鸡翅等 |
| 饮品 | 饮料、汤品 |
| 套餐 | 组合优惠 |

### 菜单排版原则

```
1. 爆品放前面
2. 价格一目了然
3. 图片+文字
4. 分类清晰
5. 留白呼吸
```

---

## 2. 外卖主图优化

### 主图检测清单

| 检查项 | 标准 |
|--------|------|
| 清晰度 | 高清无模糊 |
| 食欲感 | 色彩明亮 |
| 卖点 | 一眼看出 |
| 文字 | 价格/活动 |

### 快速优化技巧

| 问题 | 解决方法 |
|------|----------|
| 图片暗 | 调亮+对比度 |
| 没食欲 | 加暖色滤镜 |
| 看不清 | 加大文字 |
| 没特点 | 加芝士拉丝 |

## 3. 朋友圈海报

### 尺寸

| 类型 | 尺寸 |
|------|------|
| 单图 | 1:1 或 4:5 |
| 9图 | 3x3 拼图 |
| 故事 | 9:16 |

### 海报要素

```
1. 标题（吸引）
2. 主图（好看）
3. 价格（明显）
4. 活动时间（清晰）
5. 二维码/地址
```

## 4. 节日活动海报

### 常用尺寸

| 抖音封面 | 1080x1920 |

```

国庆：红色+国旗

```

## 6. 快速制作工具

### 免费工具

| 工具 | 用途 |
|------|------|
| Canva | 在线设计 |
| 创客贴 | 模板丰富 |
| 图怪兽 | 快速出图 |

### 推荐工作流

```
1. 选模板（Canva/创客贴）
2. 换图片（自己拍）
3. 改文字（店名+价格）
4. 导出（PNG/JPG）
```
