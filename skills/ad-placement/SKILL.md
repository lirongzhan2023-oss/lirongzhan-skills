## 2. 投放策略

### 预算分配

| 阶段 | 日预算 | 目的 |
|------|--------|------|
| 测试期 | 50-100元 | 测素材/人群 |
| 增长期 | 200-500元 | 扩大流量 |
| 稳定期 | 100-300元 | 维护ROI |

### 出价策略

```
新店/新品：
- 前期高于行业20%抢曝光
- 稳定后逐步降低

成熟店铺：
- 参考转化成本出价
- 低于盈亏线就调整
```

### 人群定向

| 人群 | 标签 | 出价 |
|------|------|------|
| 附近3km | 地域 | 标准 |
| 25-40岁 | 年龄 | +10% |
| 餐饮偏好 | 兴趣 | +20% |
| 复购用户 | 行为 | +30% |

## 3. 投放优化

### 核心指标

| 指标 | 公式 | 优秀值 |
|------|------|--------|
| 曝光量 | 展现次数 | 1000+/天 |
| 点击率 | 点击/曝光 | >5% |
| 转化率 | 下单/点击 | >8% |
| ROI | 营收/消耗 | >3 |

### 优化方向

| 问题 | 原因 | 解决 |
|------|------|------|
| 曝光低 | 出价低/权重低 | 提高出价、优化菜品 |
| 点击低 | 主图不吸引 | 优化主图 |
| 转化低 | 价格/活动 | 调整价格、加活动 |
| ROI低 | 成本高 | 优化人群、降低出价 |

## 4. 活动配合

### 新品上线

```
策略：集中投放
预算：日常2倍
出价：高于均值30%
目标：快速起量
```

### 促销活动

```
策略：限时放量
预算：活动期间3倍
人群：敏感价格用户
目标：提升转化
```

### 节日营销

```
春节/双11：
- 提前3天预热
- 节日当天爆发
- 预算翻倍
```

## 6. 投放流程

### 标准流程

```
1. 明确目标
   - 要曝光？转化？ROI？

2. 设置预算
   - 日预算/总预算

3. 创建计划
   - 选择产品
   - 设定出价
   - 选择人群

4. 上线测试
   - 先小预算测试

5. 数据优化
   - 根据数据调整

6. 效果复盘
   - 总结经验
```
