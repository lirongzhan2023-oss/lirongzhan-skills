# 自动化工作流技能

## 1. 定时任务

### 已配置示例

| 任务 | 时间 | 内容 |
|------|------|------|
| 每日晨报 | 每天9:00 | 天气+提醒 |
| 周报生成 | 每周一9:00 | 营收汇总 |
| 数据备份 | 每天23:00 | Excel备份 |

### Cron表达式

| 表达式 | 含义 |
|--------|------|
| 0 9 * * * | 每天9点 |
| 0 9 * * 1 | 每周一9点 |
| 0 9 1 * * | 每月1号9点 |
| 0 9 * * 0-6 | 工作日9点 |

## 2. 数据自动汇总

### 场景

```
每天需要合并：
- 美团数据
- 饿了么数据
- 抖音数据
→ 自动合并成一个Excel
```

### 实现方式

```javascript
// 伪代码：合并多平台数据
const data = {
  meituan: readExcel('美团.xlsx'),
  eleme: readExcel('饿了么.xlsx'),
  douyin: readExcel('抖音.xlsx')
};

const merged = [
  ...data.meituan,
  ...data.eleme,
  ...data.douyin
];

writeExcel('汇总.xlsx', merged);
```
