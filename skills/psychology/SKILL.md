# 心理学技能

## 核心能力

### 1. 沟通技巧

**倾听公式**：
- 👂 70%时间听
- 🗣️ 20%时间问
- 📢 10%时间说

**提问技巧**：
- **开放式**：你觉得怎么样？→ 了解想法
- **封闭式**：今天来还是明天来？→ 推动决策
- **确认式**：你是说XXX吗？→ 避免误解

**反馈技巧**：
- ❌ 不要：我觉得你这样不对
- ✅ 要说：我的建议是XXX，因为XXX

---

### 2. 客户心理

**消费决策5阶段**：
1. **注意** → 看到（门店外观、商品）
2. **兴趣** → 想知道（菜单、价格）
3. **欲望** → 想要（口味、体验）
4. **信任** → 相信（好评、推荐）
5. **行动** → 购买（下单、付款）

**价格心理**：
- 锚定效应：先展示高价，再推低价
- 尾数效应：99元比100元感觉便宜
- 套餐优惠：让人觉得自己赚了

**促单技巧**：
- 紧迫感："只剩最后X份"
- 稀缺性："这个口味限时供应"
- 社会认同："这道菜是我们店销量第一"

### 3. 销售话术

**接待话术**：
```
"欢迎光临，请问几位？"
"今天是第一次来吗？我们家招牌是XXX"
"今天想吃点清淡的还是重口的？"
```

**推荐话术**：
```
"我们新品XXX，很多老客户都在点"
"这个是我们主厨推荐的，满意度最高"
"搭配这个饮品只要+5元，很划算"
```

**处理异议**：
```
客户：太贵了
→ "我们的分量是别家的1.5倍，您看这个对比图"
客户：考虑一下
→ "好的，这是我们的菜单，您先看看，有问题随时问我"
```

### 4. 压力管理

**焦虑缓解**：
- **深呼吸**：4秒吸气 → 7秒屏住 → 8秒呼气
- **写下来**：把担心的写下来，大脑就不焦虑了
- **运动**：跑步/散步降低皮质醇

**情绪调节**：
- **暂停**：情绪上来时，先数10秒
- **换位**：对方为什么这么说？
- **聚焦**：现在能做什么？

**长期建议**：
- 每天留30分钟独处时间
- 每周至少运动2次
- 睡眠7小时以上

---

### 5. 团队管理心理

**员工激励**：
- **即时反馈**：做得好马上夸
- **成长可见**：给明确的进步目标
- **尊重需求**：听他们的想法

**批评方式**：
- ❌ "你怎么又迟到了！"
- ✅ "最近迟到2次了，是有什么困难吗？"

**开会技巧**：
- 结论先行
- 少说多听
- 鼓励不同意见
