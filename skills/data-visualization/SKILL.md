# 数据可视化技能

## 1. 图表选择

### 常用图表

| 数据类型 | 推荐图表 | 用途 |
|----------|----------|------|
| 趋势变化 | 折线图 | 营收/单量走势 |
| 对比 | 柱状图 | 品类/月份对比 |
| 占比 | 饼图/环形图 | 成本结构 |
| 排名 | 条形图 | 热销菜品 |
| 关系 | 散点图 | 客单价vs单量 |

---

## 2. Excel图表制作

### 基础图表

#### 柱状图（对比）

```
选中数据 → 插入 → 柱形图

适用：
- 每月营收对比
- 菜品销量排名
- 各成本占比
```

#### 折线图（趋势）

```
选中数据 → 插入 → 折线图

适用：
- 营收趋势
- 单量变化
- 评分走势
```

#### 饼图（占比）

```
选中数据 → 插入 → 饼图

适用：
- 成本结构
- 渠道占比
- 品类占比
```

## 3. 图表美化

### 快速美化

| 元素 | 建议 |
|------|------|
| 颜色 | 用品牌色系 |
| 字体 | 统一字体 |
| 标题 | 清晰明了 |
| 图例 | 位置统一 |

### 配色方案

```
餐饮常用：
- 暖色：红、橙、黄（食欲）
- 冷色：蓝、绿（清爽）

推荐配色：
- 主色：#FF6B35（橙红）
- 辅色：#004E89（深蓝）
- 背景：#F5F5F5（浅灰）
```

## 4. 经营看板

### 核心指标

| 指标 | 图表 | 更新频率 |
|------|------|----------|
| 今日营收 | 大数字+目标进度条 | 实时 |
| 今日单量 | 数字+趋势线 | 实时 |
| 转化率 | 数字+环形图 | 每小时 |
| 评分 | 星星+数字 | 实时 |

### 看板布局

```
┌─────────────┬─────────────┐
│   今日营收   │   今日单量  │
│   ¥10,763  │    78单    │
│   进度条    │   趋势线    │
├─────────────┼─────────────┤
│   转化率    │    评分     │
│    3.5%    │   4.7⭐    │
│   环形图    │   数字      │
└─────────────┴─────────────┘
```

## 5. 周报/月报模板

### 数据结构

```
【周报数据】
日期 | 营收 | 单量 | 客单价 | 转化率 | 评分
周一 | xxx | xxx | xxx | xxx | xxx
...
周日 | xxx | xxx | xxx | xxx | xxx

【图表建议】
1. 营收折线图（7天走势）
2. 品类销量柱状图
3. 成本结构饼图
```

### Excel制作步骤

```
1. 整理数据
2. 选中区域 → 插入图表
3. 调整类型
4. 美化（颜色、字体）
5. 添加标题
6. 另存为图片
```
