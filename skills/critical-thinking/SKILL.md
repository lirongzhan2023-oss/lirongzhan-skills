# 批判性思维技能

## 核心能力

### 1. 逻辑分析

**基础逻辑**：

| 形式 | 结构 | 示例 |
|------|------|------|
| 演绎 | 大前提→小前提→结论 | 所有人都会死，苏格拉底是人，所以... |
| 归纳 | 观察→规律→结论 | 这只天鹅是白的，那只也是，所以... |

**常见逻辑谬误**：

| 谬误 | 描述 | 示例 |
|------|------|------|
| 诉诸权威 | 权威说的就是对 | "专家说的一定对" |
| 从众 | 大家都做就是好 | "别人都买，我也买" |
| 诉诸情感 | 用感情代替理性 | "你不支持就是不爱" |
| 稻草人 | 歪曲对方观点 | "他说A，你说他支持B" |
| 因果谬误 | 相关当因果 | "冰激凌销量高时，溺水多" |
| 非黑即白 | 只有两个极端 | "不赞成就反对" |

### 2. 信息甄别

**事实vs观点**：

| 事实 | 观点 |
|------|------|
| 可验证 | 主观判断 |
| "今天气温30度" | "今天很热" |
| "月销1000单" | "生意很好" |

**信息来源判断**：

| 可靠性高 | 可靠性低 |
|----------|----------|
| 政府/官方 | 自媒体爆料 |
| 学术期刊 | 朋友圈转发 |
| 权威媒体 | 网红推荐 |
| 一手数据 | 二手解读 |

**假新闻特征**：
```
1. 来源不明
2. 情绪化标题
3. 缺乏具体细节
4. 只给一个信源
5. 正好符合预期
```

### 3. 问题拆解

**5Why分析法**：
```
问题：生意不好

为什么？→ 因为客流少
为什么客流少？→ 因为曝光低
为什么曝光低？→ 因为没做推广
为什么没做推广？→ 因为不知道怎么做
为什么不知道？→ 因为没学习

→ 解决方案：学习推广
```

**MECE原则**：
```
相互独立（Mutually Exclusive）
完全穷尽（Collectively Exhaustive）

分类时不重叠、不遗漏
```

**金字塔原理**：
```
结论先行
以上统下
归类分组
逻辑递进
```

### 4. 独立思考

**多角度分析**：

| 角度 | 问题 |
|------|------|
| 自己 | 我有什么偏见？ |
| 对方 | 他为什么这么说？ |
| 第三方 | 旁观者怎么看？ |
| 长期 | 5年后再看呢？ |
| 反方 | 反对的理由是什么？ |

**避免认知偏差**：

| 偏差 | 描述 | 对策 |
|------|------|------|
| 确认偏误 | 只看想看的 | 主动找反例 |
| 锚定效应 | 第一次信息影响判断 | 先独立思考 |
| 后见之明 | 事后认为必然 | 记录事前预测 |
| 幸存者偏差 | 只看成功的 | 分析失败的 |
| 过度自信 | 高估自己能力 | 数据说话 |

---

### 5. 决策质量

**决策检查清单**：

```
□ 我是基于事实还是感觉？
□ 信息来源可靠吗？
□ 有什么重要信息我忽略了？
□ 反对意见是什么？
□ 最坏情况我能接受吗？
□ 10个月后我会后悔吗？
□ 别人做会怎么做？
```

**快速决策法**：

| 场景 | 方法 |
|------|------|
| 信息不全 | 快速试错、小步迭代 |
| 选择困难 | 两选一，抛硬币辅助 |
| 重大决定 | 24小时冷静期 |
| 可逆决定 | 先做再说 |
| 不可逆决定 | 充分论证 |
