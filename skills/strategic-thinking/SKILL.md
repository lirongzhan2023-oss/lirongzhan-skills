# 战略思维技能

## 核心能力

### 1. 战略规划

**战略制定流程**：
```
1. 愿景 → 3-5年后想成为什么样？
2. 现状 → 现在在哪里？有什么？
3. 差距 → 愿景和现状的差距
4. 路径 → 怎么填补差距？
5. 资源 → 需要什么支持？
6. 里程碑 → 关键节点是什么？
```

**战略层次**：

| 层次 | 问题 | 示例 |
|------|------|------|
| 公司层 | 做什么业务？ | 从堂食扩展到外卖 |
| 业务层 | 怎么竞争？ | 差异化定位 |
| 执行层 | 怎么做？ | 菜单优化、活动策划 |

### 2. 决策分析

**利弊分析法**：
```
决策：XXX

【利】
1. XXX
2. XXX

【弊】
1. XXX
2. XXX

【结论】：做/不做
```

**10/10/10法则**：
```
这个决定：
- 10分钟后，我会怎么想？
- 10个月后，我会怎么想？
- 10年后，我会怎么想？
```

**概率思维**：
```
- 最好情况：XX%（会发生什么）
- 最可能情况：XX%
- 最坏情况：XX%（我能承受吗？）
```

### 3. 竞争分析

**SWOT分析**：

| | 有利 | 不利 |
|---|---|---|
| **内部** | 优势S<br>1. XXX<br>2. XXX | 劣势W<br>1. XXX<br>2. XXX |
| **外部** | 机会O<br>1. XXX<br>2. XXX | 威胁T<br>1. XXX<br>2. XXX |

**波特五力**：
1. 现有竞争者
2. 潜在进入者
3. 替代品
4. 供应商议价能力
5. 客户议价能力

**差异化定位**：
```
【对手】：XXX
【对手优势】：XXX
【我的差异化】：
- 产品差异化
- 服务差异化
- 价格差异化
- 渠道差异化
```

### 4. 风险管理

**风险识别**：

| 风险类型 | 风险项 | 影响 | 概率 |
|----------|--------|------|------|
| 运营 | 食材断供 | 高 | 中 |
| 市场 | 竞品降价 | 中 | 高 |
| 财务 | 现金流断 | 高 | 低 |

**应对策略**：

| 策略 | 适用场景 |
|------|----------|
| 规避 | 高风险、不值得做 |
| 减轻 | 降低发生概率或影响 |
| 转移 | 保险、合作伙伴 |
| 接受 | 低风险、影响小 |

**最坏情况预案**：
```
如果最坏情况发生：
1. XXX
2. XXX

我还能活多久？→ 现金流能撑X个月
```

### 5. 思维框架

**第一性原理**：
```
从最基础的真理出发，而不是类比

❌ 类比：别人做XX成功了，我也做
✅ 第一性原理：用户需要XX，我提供XX的方式成本最低
```

**10/10/10法则**：
```
情绪vs理性短期vs长期

问自己：
- 10分钟后感受？
- 10个月后影响？
- 10年后回头看？
```

**系统思维**：
```
看整体 > 看部分
看关系 > 看要素
看长期 > 看短期

找杠杆点：哪里改变一点，能带动全局？
```
