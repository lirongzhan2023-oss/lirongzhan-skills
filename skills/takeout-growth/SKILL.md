# 外卖增长技能

## 1. 数据诊断专家

### 读图流程（必须严格遵循）

**第一步：全图扫描**
```
□ 店名/品牌名
□ 日期/时间周期
□ 平台名称（美团/饿了么/抖音）
□ 核心指标区（通常在顶部/中间）
□ 小字/备注/注释
□ 趋势箭头
□ 页面底部信息
```

**第二步：数据提取**
```
【必提取数据】
- 单量：____单
- 收入：____元
- 访客数：____人
- 转化率：____%
- 复购率：____%
- 评分：____
- 客单价：____元（计算验证）
```

**第三步：数据验证**
```
□ 客单价 = 收入 ÷ 单量
□ 转化率 = 订单 ÷ 访客 × 100%
□ 数据逻辑一致
□ 数值在合理范围
```

**第四步：行业对标**
```
【头部商家基准】
- 转化率：5-8%
- 复购率：25-35%
- 评分：4.9
- 客单价：32-40元
```

## 2. 转化率优化

### 2.1 主图优化

**主图检测清单**：

| 检查项 | 标准 | 问题诊断 |
|--------|------|----------|
| 图片清晰度 | 高清无模糊 | ❌模糊→点击低 |
| 食欲感 | 色彩明亮、有食欲 | ❌灰暗→无兴趣 |
| 卖点突出 | 一眼看出是什么 | ❌看不出→不点 |
| 文字信息 | 品牌名+特色 | ❌没有→没记忆点 |

**主图优化方向**：
```
1. 菜品实拍（不用网图）
2. 光线充足（暖色灯光）
3. 角度45度（俯拍）
4. 芝士拉丝特写（针对焗饭）
5. 加上价格/活动标签
```

---

### 2.2 菜单优化

**菜单结构**：

| 位置 | 内容 | 目的 |
|------|------|------|
| 前3个 | 爆品/引流款 | 快速下单 |
| 4-6个 | 利润款/套餐 | 提升客单 |
| 7-10个 | 补充款 | 满足长尾 |
| 最后 | 饮品/小食 | 关联销售 |

**优化要点**：
```
□ 爆品置顶
□ 分类清晰（主食/小吃/饮品）
□ 套餐组合
□ 商品描述有吸引力
□ 规格设置合理
```

---

### 2.3 活动优化

**满减设计**：

| 门槛 | 减额 | 目的 |
|------|------|------|
| 20-3 | 低门槛 | 拉新 |
| 50-8 | 中门槛 | 提客单 |
| 80-15 | 高门槛 | 大单 |

**活动组合**：
```
【头部商家标配】
- 满减：20-3, 40-6, 60-10
- 折扣商品：2-3款引流
- 店铺券：满30-5
- 会员红包：复购专用
```

## 3. 复购率提升

### 3.1 会员体系

**会员等级设计**：

| 等级 | 条件 | 权益 |
|------|------|------|
| 普通 | 注册 | 基础优惠 |
| 银卡 | 消费满200 | 9折 |
| 金卡 | 消费满500 | 8.5折+优先 |
| 黑卡 | 消费满1000 | 8折+专属 |

**积分体系**：
```
- 消费1元 = 1积分
- 100积分 = 5元券
- 积分当钱花
```

---

### 3.2 私域沉淀

**加微路径**：
```
好评返券 → 添加微信 → 进群 → 会员绑定
```

**社群运营**：

| 内容 | 目的 |
|------|------|
| 新品优先 | 激活 |
| 专属折扣 | 留存 |
| 复购提醒 | 触达 |
| 抽奖互动 | 活跃 |

## 4. 爆品打造

### 4.1 选品原则

**爆品特征**：
```
1. 受众广（多数人爱吃）
2. 毛利高（40%以上）
3. 复购强（吃了还想吃）
4. 出品快（不影响效率）
5. 差异化（有记忆点）
```

---

### 4.2 爆品定价

**引流款**：
```
- 不赚钱或少赚钱
- 目的是吸引流量
- 定价在成本×1.2
```

**利润款**：
```
- 主要收入来源
- 定价在成本×3
- 搭配销售
```

---

### 4.3 流量获取

**爆品流量组合**：
```
1. 平台活动：秒杀/特价
2. 主图推荐：置顶+标签
3. 好评引导：形成口碑
4. 复购锁定：专属优惠
```
