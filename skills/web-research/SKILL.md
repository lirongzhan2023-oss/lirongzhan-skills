# 网上资料查询技能

## 核心能力

### 1. 行业信息搜集

- 行业报告：市场规模、增长趋势、竞争格局
- 品类数据：消费者偏好、价格区间、渠道分布
- 趋势分析：新兴品类、热点方向

### 2. 竞品调研

- 产品分析：功能、价格、包装
- 营销活动：促销策略、推广渠道
- 用户评价：好评、差评、改进方向

### 3. 知识查询

- 概念定义：专业术语解释
- 方法论：分析框架、运营模型
- 案例参考：成功/失败案例

### 4. 新闻资讯

- 行业动态：新品上市、融资并购
- 政策变化：法规解读、影响分析
- 热点事件：舆论分析、应对建议

### 5. 数据验证

- 多源比对：至少2个来源交叉验证
- 置信度判断：权威来源/一般来源/待核实
- 信息时效：标注时间戳

## 查询原则

### 信息质量排序
1. **权威来源**：政府统计、行业协会、知名咨询公司
2. **一手数据**：财报、招股书、官方公告
3. **新闻报道**：主流媒体报道
4. **网络信息**：一般文章、网友分享（需核实）

### 必做事项
- 注明信息来源（来源 + 时间）
- 判断信息时效性
- 多源交叉验证
- 区分事实与观点

### 严禁事项
- 不确定的信息当作确定事实
- 使用未经核实的网传数据
- 忽略信息的时间因素

## 输出格式

```
【查询主题】：XXX

【关键发现】
1. XXX（来源1，时间）
2. XXX（来源2，时间）

【数据验证】
- 来源A：XXX → ✅ 已验证
- 来源B：XXX → ✅ 已验证

【参考链接】
- 来源1：URL
- 来源2：URL
```

## 餐饮行业常用查询

- 品类报告：艾瑞、尼尔森、美团研究院
- 竞品分析：大众点评、小红书、抖音
- 供应链：食材价格、供应商信息
- 政策法规：食品安全、外卖平台规则
