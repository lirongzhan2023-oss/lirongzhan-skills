# 演示汇报技能

## 核心能力

### 1. 演讲表达

**演讲结构**：

| 部分 | 时长 | 内容 |
|------|------|------|
| 开场 | 10% | 抓注意力 |
| 主体 | 80% | 核心内容 |
| 结尾 | 10% | 行动号召 |

**开场技巧**：
```
1. 提问："你有没有想过..."
2. 故事："我遇到过一件事..."
3. 数据："过去一年，XX增长了..."
4. 震惊："90%的人不知道..."
```

**结尾技巧**：
```
1. 总结："今天我们讲了3点..."
2. 行动："接下来大家可以..."
3. 愿景："想象一下5年后..."
4. 引用："最后用一句话总结..."
```

**控场技巧**：
```
- 眼神交流：照顾全场
- 停顿：重要观点前停顿
- 互动：提问、举手
- 紧张怎么办？→ 慢下来、深呼吸
```

### 2. PPT制作

**PPT结构**：

| 页数 | 内容 | 说明 |
|------|------|------|
| 1 | 封面 | 主题+名字 |
| 2 | 目录 | 大纲 |
| 3-N | 主体 | 每页一个观点 |
| 最后 | 总结 | 行动号召 |

**每页结构**：
```
标题：核心观点（一句话）
正文：支撑论据（3点以内）
图表：可视化数据
```

**视觉原则**：
```
1. 少字多图
2. 对齐整齐
3. 颜色统一
4. 留白呼吸
```

**字体建议**：
- 标题：36pt+
- 正文：24pt+
- 衬线体（标题）+无衬线体（正文）

### 3. 商业计划书

**BP结构**：

| 章节 | 内容 |
|------|------|
| 1. 项目概述 | 一句话介绍 |
| 2. 市场分析 | 痛点+机会 |
| 3. 产品/服务 | 解决方案 |
| 4. 商业模式 | 怎么赚钱 |
| 5. 运营数据 | 成绩单 |
| 6. 团队 | 谁来做 |
| 7. 融资计划 | 需要多少钱 |
| 8. 发展规划 | 未来计划 |

**BP要点**：
```
- 痛点够痛
- 解决方案够清晰
- 数据有说服力
- 团队有经验
- 财务预测合理
```

### 5. 说服技巧

**说服三要素**：

| 要素 | 方法 | 示例 |
|------|------|------|
| 逻辑 | 数据+推理 | "数据显示..." |
| 情感 | 故事+共鸣 | "我遇到过..." |
| 信誉 | 权威+背书 | "专家说..." |

**故事框架**：
```
1. 背景：当时什么情况
2. 困难：遇到什么挑战
3. 行动：我做了什么
4. 结果：取得了什么
5. 感悟：学到了什么
```

**处理反对**：
```
"但是..."
→ "你的顾虑我理解，其实..."
"太贵了"
→ "和XX对比，其实我们..."
"不需要"
→ "那您觉得什么更重要？"
```
