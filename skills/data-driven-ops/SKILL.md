# 数据驱动运营技能

## 1. 数据源整合

### 门店数据

| 数据 | 来源 | 用途 |
|------|------|------|
| 营收 | Excel/导出 | 基础指标 |
| 成本 | Excel/导出 | 利润分析 |
| 订单 | 外卖后台 | 转化分析 |
| 评分 | 平台 | 质量指标 |

### 行业数据

| 数据 | 来源 | 用途 |
|------|------|------|
| 毛利率 | 行业报告 | 对标 |
| 转化率 | 行业报告 | 目标设定 |
| 趋势 | 公开报告 | 方向判断 |

### 地理数据

| 数据 | 来源 | 用途 |
|------|------|------|
| 门店位置 | 高德API | 基础数据 |
| 周边商家 | 高德API | 竞争分析 |
| 客户距离 | 高德API | 配送范围 |

---

## 2. 数据应用场景

### 场景1：选址分析

```
输入：候选地址
处理：
1. 高德API获取坐标
2. 搜索周边3km餐饮数量
3. 分析品类分布
4. 评估竞争强度

输出：推荐指数（0-100分）
```

### 场景2：定价策略

```
输入：成本、竞品价格
处理：
1. 计算毛利目标
2. 分析竞品价格带
3. 确定定价区间

输出：建议价格
```

### 场景3：运营优化

```
输入：门店数据
处理：
1. 对标行业数据
2. 找出差距
3. 制定改进方案
4. 追踪效果

输出：改进清单
```

## 7. 持续迭代

### PDCA循环

```

Do：执行落地
Check：数据验证
Act：优化调整
```
