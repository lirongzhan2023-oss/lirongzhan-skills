# 推广宣传技能

## 核心能力

### 1. 线上推广

**朋友圈推广**：

| 内容类型 | 频率 | 目的 |
|----------|------|------|
| 菜品图 | 每天1-2条 | 吸引 |
| 活动信息 | 每周2-3条 | 转化 |
| 客户好评 | 每天1条 | 信任 |
| 日常分享 | 每天1条 | 亲近 |

**发布时间**：
- 早餐：7:00-9:00
- 午餐：11:00-13:00
- 下午：15:00-17:00
- 晚餐：17:00-19:00
- 夜宵：20:00-22:00

**抖音推广**：

| 类型 | 时长 | 内容 |
|------|------|------|
| 菜品展示 | 15-30秒 | 高清美食 |
| 后厨日常 | 30-60秒 | 真实 |
| 挑战赛 | 15秒 | 互动 |
| 探店视频 | 1-3分钟 | 种草 |

**小红书推广**：

| 类型 | 结构 |
|------|------|
| 种草笔记 | 痛点+体验+推荐 |
| 探店攻略 | 地址+人均+亮点 |
| 食谱分享 | 教程+成品 |

### 2. 线下推广

**地推策略**：

| 地点 | 目标 | 方式 |
|------|------|------|
| 写字楼 | 午市客流 | 传单+扫码 |
| 社区 | 家庭客 | 入户券 |
| 商圈 | 年轻人 | 举牌+扫码 |
| 超市 | 周边居民 | 堆头+试吃 |

**异业合作**：

| 合作方 | 形式 | 目的 |
|--------|------|------|
| 电影院 | 票根优惠 | 互相引流 |
| KTV | 套餐合作 | 聚会场景 |
| 健身房 | 运动后套餐 | 健康餐饮 |
| 幼儿园 | 亲子套餐 | 家庭客 |

**社区推广**：

```
1. 社区群：加入业主群，发优惠券
2. 社区公告栏：贴海报
3. 物业合作：快递柜广告
4. 社区活动：赞助义卖
```

### 3. 平台推广

**美团/饿了么投流**：

| 投放类型 | 适用 | 出价 |
|----------|------|------|
| 揽客宝 | 新店期 | 2-5元/次 |
| 推广通 | 日常 | 1-3元/点击 |
| 会员红包 | 复购 | 2-5元/券 |

**投流技巧**：
```
1. 新店期：先冲销量
2. 稳定期：优化转化
3. 旺季：加大投放
4. 淡季：维持曝光
```

**平台活动报名**：

| 活动 | 效果 | 门槛 |
|------|------|------|
| 秒杀 | 拉新 | 低价限量 |
| 满减 | 提客单 | 门槛合理 |
| 免配送 | 转化 | 成本控制 |
| 品牌馆 | 品牌 | 需达标 |

### 4. 私域推广

**社群运营**：

| 社群类型 | 内容 | 目的 |
|----------|------|------|
| 福利群 | 优惠信息 | 转化 |
| 会员群 | 专属福利 | 留存 |
| 活动群 | 主题活动 | 互动 |

**群内容规划**：

```
早上：天气预报+早餐推荐
中午：午市套餐+优惠
下午：下午茶+新品
晚上：晚餐推荐+好评
```

**公众号推广**：

| 类型 | 频率 | 内容 |
|------|------|------|
| 推文 | 每周1篇 | 活动+品牌 |
| 菜单 | 每月 | 新品 |
| 节日 | 节日 | 祝福+优惠 |

### 5. 品牌建设

**品牌定位**：

```
【我是谁】：XXX
【卖给谁】：XXX
【解决什么】：XXX
【有什么不同】：XXX
```

**品牌故事**：

```
模板：
1. 初心：为什么做？
2. 坚持：做了什么？
3. 成就：现在怎样？
```

**品牌口号**：

| 类型 | 示例 |
|------|------|
| 卖点型 | XXX，就选XXX |
| 情感型 | XXX，让生活更XXX |
| 行动型 | XXX，就去XXX |

**VI元素**：

| 元素 | 内容 |
|------|------|
| Logo | 图形+文字+颜色 |
| 口号 | 一句话说明 |
| 色调 | 主色+辅色 |
| 风格 | 统一视觉 |

## 推广预算

**推广占比参考**：

| 阶段 | 占比 | 说明 |
|------|------|------|
| 新店期 | 10-15% | 投入期 |
| 成长期 | 5-10% | 稳定期 |
| 成熟期 | 3-5% | 维持期 |

**渠道分配**：

| 渠道 | 占比 | 效果 |
|------|------|------|
| 线上（抖音/小红书） | 40% | 品牌 |
| 外卖平台 | 30% | 销售 |
| 私域 | 20% | 复购 |
| 线下 | 10% | 覆盖 |
