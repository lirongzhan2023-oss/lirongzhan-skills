# 时间管理技能

## 核心方法

### 1. GTD（Getting Things Done）

**核心原则**：清空大脑，把所有事情记下来

**5步流程**：

1. **收集**（Capture）
   - 把所有想到的事写下来
   - 来源：脑中想法、邮件、微信、安排

2. **处理**（Process）
   - 每个任务问：2分钟内能做完？
   - 是→马上做
   - 否→下一步

3. **整理**（Organize）
   - 分类：
     - ✅ 等待（别人完成）
     - ⏰ 稍后做（有时间再做）
     - 📅 特定时间做
     - 📋 项目（复杂任务拆解）

4. **回顾**（Review）
   - 每天/每周回顾
   - 更新任务列表
   - 删除已完成/不再需要

5. **执行**（Engage）
   - 依据优先级执行
   - 相信系统，不焦虑

### 2. 番茄工作法

**原理**：25分钟专注 + 5分钟休息

**操作**：
1. 选一个任务
2. 设定25分钟计时
3. 专注工作，不打断
4. 计时结束，休息5分钟
5. 每4个番茄钟，长休息15-30分钟

**适用场景**：
- 写方案、算账、回复消息
- 不适合：突发沟通、接单高峰期

### 3. 重要/紧急矩阵

| | 紧急 | 不紧急 |
|---|---|---|
| **重要** | 🔴 马上做<br>（危机、客户投诉） | 🟡 计划做<br>（规划、学习、预防） |
| **不重要** | 🟠 委托做<br>（琐事、临时请求） | ⚪ 尽量不做<br>（刷手机、无效社交） |

**餐饮老板常见问题**：
- ❌ 陷入紧急不重要（不断接单、应付检查）
- ✅ 每天留时间做重要不紧急（新品研发、员工培训）

### 4. 日/周/月计划

**日计划**：
```
📅 今天必须完成（3件）：
1. XXX
2. XXX
3. XXX

⏰ 时间块：
- 9:00-11:00：XXX
- 14:00-16:00：XXX
- 19:00-20:00：XXX
```

**周计划**：
```
📅 本周目标：
- 销售：XX元（环比+X%）
- 重点：XXX

🗓️ 周一：XXX
周二：XXX
...
周日：XXX（复盘）
```

**月计划**：
```
📅 本月核心目标：
1. 销售：XX万
2. 利润：XX万
3. 重点：XXX

🔑 关键结果：
- 新品上市：X款
- 活动：X场
- 招聘：X人
```

### 5. 精力管理

**黄金时间**：
- 🧠 **脑力高峰**：上午9-11点 → 决策、算账、规划
- 💪 **体力高峰**：中午 → 出餐、备货
- 📞 **沟通高峰**：下午2-4点 → 开会、招聘
- 📊 **复盘高峰**：晚上 → 数据分析、计划

**建议**：
- 早餐后先做最重要的决定
- 下午2点前处理完所有"思考型"工作
- 晚上不复盘太晚（影响睡眠）
