# 投资理财技能

## 核心能力

### 1. 理财基础

**理财金字塔**：

```
           投资
         /      \
      储蓄    保险
        \    /
        工资
```

**应急基金**：
```
目标：3-6个月生活费

计算：
月支出 × 3~6 = 应急基金目标

存放：活期/货币基金（随时可取）
```

**预算管理**：
```
收入 - 储蓄 = 支出

50/30/20法则（参考）：
- 50% 必要支出（房租、吃饭、交通）
- 30% 可选支出（娱乐、购物）
- 20% 储蓄投资
```

---

### 2. 投资常识

**投资品种**：

| 类型 | 风险 | 收益 | 流动性 |
|------|------|------|--------|
| 存款 | 低 | 2-3% | 高 |
| 货币基金 | 低 | 2-4% | 高 |
| 债券 | 中 | 4-6% | 中 |
| 基金 | 中高 | 5-15% | 高 |
| 股票 | 高 | -50%~+50% | 高 |
| 房产 | 高 | 3-8% | 低 |

**收益原则**：
- 高收益 = 高风险
- 流动性 = 收益折价
- 分散 = 降低风险

**定投**：
```
定期投入固定的金额

优点：
- 摊平成本
- 不需要择时
- 培养储蓄习惯

示例：每月投入1000元买指数基金
```

### 3. 餐饮投资

**开店投资模型**：

| 投资项 | 金额 | 占比 |
|--------|------|------|
| 装修 | 5-15万 | 30% |
| 设备 | 3-8万 | 20% |
| 首季租金 | 2-5万 | 15% |
| 首季人工 | 3-6万 | 20% |
| 首季原材料 | 2-4万 | 10% |
| 流动资金 | 1-2万 | 5% |
| **合计** | **16-40万** | 100% |

**回本周期**：

| 业态 | 回本周期 | 月利润目标 |
|------|----------|------------|
| 快餐 | 6-12个月 | 1-2万 |
| 正餐 | 12-18个月 | 2-3万 |
| 茶饮 | 6-9个月 | 1.5-2.5万 |

**风险控制**：
```
1. 不借钱开店
2. 预留6个月生活费
3. 做好最坏打算（亏完不影响生活）
4. 单店盈利后再开第二家
```

### 4. 资产配置

**简易配置**：

| 年龄 | 股票比例 | 债券比例 | 现金比例 |
|------|----------|----------|----------|
| 30岁 | 80% | 15% | 5% |
| 40岁 | 70% | 25% | 5% |
| 50岁 | 50% | 40% | 10% |

**4321法则（参考）**：
```
40% 投资（股票、基金、房产）
30% 生活开支
20% 储蓄（应急、养老）
10% 保险
```

**分散原则**：
- 品类分散：股票+债券+商品
- 地域分散：国内+海外
- 时间分散：定投

---

### 5. 财务安全

**保险配置**：

| 保险 | 作用 | 建议 |
|------|------|------|
| 医疗险 | 报销医疗费 | 优先配置 |
| 重疾险 | 生病补偿 | 确诊即赔 |
| 寿险 | 家庭责任 | 有贷款/孩子必买 |
| 意外险 | 意外伤残 | 必买 |

**债务管理**：
```
好债 vs 坏债

好债：
- 投资自己（学费）
- 投资资产（房产、生意）

坏债：
- 消费负债（信用卡分期）
- 高利贷

原则：负债率 < 50%
```
