# Excel读取与分析技能

## 核心能力

### 1. 文件读取

**支持格式**：
- .xlsx (Excel 2007+)
- .xls (Excel 2003)
- .csv (逗号分隔值)

**读取方法**：

```javascript
// Node.js + xlsx
const XLSX = require('xlsx');

// 读取文件
const wb = XLSX.readFile('文件路径.xlsx');

// 获取所有工作表
console.log(wb.SheetNames);

// 读取指定工作表
const ws = wb.Sheets['工作表名'];

// 转换为JSON
const data = XLSX.utils.sheet_to_json(ws);
```

**Python方法**：
```python
import pandas as pd

# 读取Excel
df = pd.read_excel('文件.xlsx', sheet_name=None)
# sheet_name=None 读取所有工作表

# 读取CSV
df = pd.read_csv('文件.csv')
```

### 2. 餐饮财务数据分析

**核心指标提取**：

| 指标 | 提取方式 | 正常范围 |
|------|----------|----------|
| 营收 | 当月营收字段 | - |
| 物料成本 | 总物料使用 | <50% |
| 人工成本 | 人事费用 | 20-25% |
| 租金 | 门市租金 | 5-10% |
| 推广费 | 推广费用 | 3-5% |
| 毛利率 | 1-物料占比 | >40% |
| 净利率 | 纯利润/营收 | >10% |

---

### 3. 问题诊断规则

**红灯指标**：

| 指标 | 红灯线 | 问题 |
|------|--------|------|
| 物料成本占比 | >55% | 物料浪费/采购价高 |
| 人工成本占比 | >30% | 人效低/人员多 |
| 净利率 | <5% | 盈利困难 |
| 推广费占比 | <1% | 流量不足 |

---

### 4. 输出格式

**数据提取输出**：
```
【文件】：XXX.xlsx
【工作表】：['汇总', '薪资表', ...]

【关键数据】：
| 指标 | 金额 | 占比 |
|------|------|------|
| 营收 | XX元 | 100% |
| 成本 | XX元 | XX% |

```

```

1. XXX

1. XXX
2. XXX
```

## 常用分析模板

### 月度营收分析

### 利润分析

- 净利率
- 各环节成本占比
