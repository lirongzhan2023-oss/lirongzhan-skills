# 会员体系技能

## 1. 会员等级设计

### 等级体系

| 等级 | 门槛 | 权益 | 占比 |
|------|------|------|------|
| 普通会员 | 注册 | 积分倍率1x | 60% |
| 银卡会员 | 消费500 | 积分1.5x + 9折 | 25% |
| 金卡会员 | 消费2000 | 积分2x + 8.5折 | 12% |
| 黑卡会员 | 消费5000 | 积分3x + 8折 + 专属 | 3% |

### 升级规则

```
升级条件：
- 累计消费金额
- 或累计订单数
- 或累计积分

保级条件：
- 年度消费满XXX
- 否则降级
```

## 2. 积分体系

### 积分规则

| 行为 | 积分 |
|------|------|
| 消费1元 | 1积分 |
| 好评 | +50积分 |
| 邀请好友 | +100积分 |
| 生日月 | +200积分 |

### 积分兑换

| 积分 | 礼品 |
|------|------|
| 100 | 5元券 |
| 300 | 15元券 |
| 500 | 30元券 |
| 1000 | 招牌菜1份 |

---

## 3. 会员权益

### 权益矩阵

| 权益 | 普通 | 银卡 | 金卡 | 黑卡 |
|------|------|------|------|------|
| 折扣 | 无 | 9折 | 8.5折 | 8折 |
| 积分 | 1x | 1.5x | 2x | 3x |
| 生日礼 | 无 |  coupon |  coupon+礼品 |  coupon+蛋糕 |
| 专属活动 | 无 | 有 | 有 | 有 |
| 优先售后 | 无 | 无 | 有 | 有 |

### 权益设计原则

```
1. 等级越高，权益越多
2. 权益要有感知价值
3. 核心权益：折扣+积分+专属
4. 差异化权益区分等级
```

| 活动 | 时间 | 内容 |
|------|------|------|
| 会员日 | 每周三 | 8折 |

### 节日活动

```
春节：满500抽奖
情人节：情侣套餐半价
会员周年庆：积分翻倍
```

## 5. 复购策略

### 召回体系

| 状态 | 定义 | 召回策略 |
|------|------|----------|
| 活跃 | 30天内消费 | 正常维护 |
| 沉睡 | 30-60天未消费 | 送券召回 |
| 流失 | 60天+未消费 | 大额券+专属活动 |

### 召回话术

```
【沉睡召回】
"您有50元券未领取，快来使用吧！"

【流失召回】
"想您了！送您一张100元无门槛券，欢迎回家~"
```

## 6. 数据分析

### 核心指标

| 指标 | 公式 | 目标 |
|------|------|------|
| 会员数 | 注册人数 | 1000+ |
| 活跃率 | 活跃/总数 | >50% |
| 复购率 | 复购/购买 | >30% |
| 会员贡献 | 会员营收/总营收 | >60% |
| 客单价 | 会员消费/订单 | 高于非会员20% |

### 数据看板

```
【会员数据日报】
- 新增会员：XX人
- 活跃会员：XX人
- 今日订单：XX单
- 会员消费：XX元

【周趋势】
- 活跃率：XX%
- 复购率：XX%
```
