# Excel表格优化技能

## 1. 公式优化

### 自动汇总公式

| 公式 | 用途 | 示例 |
|------|------|------|
| SUM | 求和 | =SUM(A1:A10) |
| SUMIF | 条件求和 | =SUMIF(A:A,">0",B:B) |
| AVERAGE | 平均值 | =AVERAGE(A1:A10) |
| COUNT | 计数 | =COUNT(A1:A10) |
| IF | 条件判断 | =IF(A1>100,"高","低") |
| VLOOKUP | 查找 | =VLOOKUP(A1,表,2,0) |

### 动态汇总示例

```
【原始数据】（不动）
A列：日期
B列：营收

【新增汇总列】（公式）
C1：=SUM(B:B) → 总营收
C2：=AVERAGE(B:B) → 平均营收
C3：=SUMIF(A:A,"2026-03",B:B) → 3月营收
```

### KPI仪表盘公式

```
【转化率】=B2/B1
【达标】=IF(C1>5%,"✅","⚠️")

【毛利率】=(B2-B3)/B2
【预警】=IF(D1<40%,"🔴低","🟢正常")
```

## 2. 条件格式

### 颜色预警

| 规则 | 公式 | 颜色 |
|------|------|------|
| 高于目标 | >100% | 绿色 |
| 低于目标 | <80% | 红色 |
| 中等 | 80-100% | 黄色 |

### 设置方法

```
选中区域 → 开始 → 条件格式 → 新建规则
→ 使用公式确定要设置格式的单元格
→ 格式 → 填充颜色
```

### 进度条

```
选中数值列 → 条件格式
→ 数据条 → 选择颜色
→ 自动生成进度条
```

### 图标集

```
选中列 → 条件格式 → 图标集
→ 方向箭头：↑↓
→ 彩色标志：🟢🟡🔴
```

## 3. 数据验证

### 下拉菜单

```
设置方法：
选中单元格 → 数据 → 数据验证
→ 允许：序列
→ 来源：A1:A5（下拉选项）

用途：
- 门店选择
- 状态标记
- 分类筛选
```

### 自动填充

```
A列输入门店 → 自动填充门店代码
=IF(A1="育德","Y",IF(A1="东郊","D",""))

```

## 4. 图表绑定

### 动态图表

```
1. 选中数据 → 插入图表
2. 右键图表 → 选择数据
3. 添加数据系列名称 =Sheet!$A$1
4. 调整数据区域为动态范围
```

### 切片器（推荐）

```
插入 → 切片器
→ 连接数据源
→ 可点击筛选

好处：
- 无需公式
- 可视化操作
- 动态更新
```

### 组合图

```
适用场景：
- 柱状图（营收）+折线图（增长率）

设置：
选中数据 → 插入 → 组合图
→ 勾选"次坐标轴"
```

## 6. 实战案例

### 育德店成本表优化

**原始数据**（不动）：
```
| 日期 | 物料 | 人力 | 租金 |
|------|------|------|------|
| 3/1 | 2000 | 800 | 300 |
```

**优化后**（新增公式和格式）：

```
| 日期 | 物料 | 人力 | 租金 | 总成本 | 占比 | 预警 |
|------|------|------|------|--------|------|------|
| 3/1 | 2000 | 800 | 300 | =SUM(B2:D2) | =E2/10000 | =IF(F2<50%,"🔴","🟢") |
```

**可视化**：
- E列：数据条（总成本趋势）
- F列：条件格式颜色
- G列：图标预警
- 切片器：按月份筛选
