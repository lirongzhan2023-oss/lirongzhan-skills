# 自我提升实用方法论

## 高效学习方法

### 费曼学习法
1. 选择一个概念
2. 把它教给完全不懂的人
3. 发现讲不清的地方，回过头重新学
4. 简化语言，确保能讲明白

### 刻意练习
- 每次练习聚焦在舒适区边缘
- 获得即时反馈
- 大量重复 + 持续调整

### 间隔重复
- 学习后24小时内复习
- 之后按指数增长间隔复习（1天、3天、7天、15天...）
- 使用Anki等工具辅助

## 习惯养成公式

**习惯 = 触发 + 行动 + 奖励 + 重复**

- **触发**：时间/地点/情绪/他人 → 提示你该行动
- **行动**：最小可行动作（降低门槛）
- **奖励**：即时正向反馈（身心愉悦）
- **重复**：形成自动化神经通路
