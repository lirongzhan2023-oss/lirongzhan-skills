# 内容写作技能

## 核心能力

### 1. 朋友圈文案

**结构公式**：
```
开头：抓住注意力（痛点/悬念/数字）
中间：价值输出（卖点/故事/证据）
结尾：行动引导（点赞/评论/到店）
```

**餐饮朋友圈类型**：

| 类型 | 结构 | 示例 |
|------|------|------|
| 新品推荐 | 亮点+体验+引导 | "终于上新了！XXX..." |
| 活动促销 | 限时+优惠+紧迫 | "仅限3天，全场8折..." |
| 客户好评 | 截图+感受+感谢 | "客户说..." |
| 日常分享 | 场景+情绪+互动 | "今天厨房发生..." |

**技巧**：
- 文字不宜太多，控制在100字内
- 用表情符号断句
- 配图要精美（菜品图>环境图>空镜）
- 发布时间：早11:30-13:00，晚17:30-19:30

### 2. 公众号推文

**结构模板**：

**模板1：新品发布**
```
【标题】：XXX终于来了！
【引言】：等待XX天，因为我们坚持...
【正文】：
- 研发故事
- 产品亮点（3点）
- 试吃体验
【结尾】：XX门店有售/点击小程序
```

**模板2：活动宣传**
```
【标题】：XX活动｜省XX钱
【引言】：好消息！XX活动开始...
【正文】：
- 活动内容
- 参与方式
- 时间地点
【结尾】：名额有限，先到先得
```

**模板3：品牌故事**
```
【标题】：XX背后的故事
【引言】：从XX到XX，我们走了X年
【正文】：
- 初心
- 困难与坚持
- 现在的我们
【结尾】：感谢陪伴，欢迎到店
```

### 3. 外卖店铺文案

**商品描述模板**：
```
【名称】：XXX（主关键词）
【描述】：XX做法，XX口感，XX特色
【建议】：XX人群必点/配XX更佳
【注意】：XX口味可选/XX克/人份
```

**示例**：
> 芝士焗饭｜经典招牌
> 进口马苏里拉芝士，拉丝满满！粒粒分明的东北大米配上秘制酱汁，大口吃才过瘾。🍕
> 👨‍👩‍👧‍👦 一人食首选，配饮品更优惠！

**店铺简介**：
```
XX年专注XX
📍 门店：XX、XX
🏆 荣誉：XX
💬 理念：XX
```

### 5. 小红书/抖音文案

**爆款结构**：

**美食推荐**：
```
【开头】：XXX也太好吃了吧！| 私藏已久的宝藏店
【中间】：
- 环境（氛围感）
- 菜品（色香味）
- 性价比
【结尾】：店铺地址/人均/推荐指数⭐⭐⭐⭐⭐
```

**探店vlog**：
```
【标题】：XX人均30+，吃到扶墙出
【脚本】：
- 进店点单
- 等餐过程
- 开吃镜头（特写）
- 总结评价
```
