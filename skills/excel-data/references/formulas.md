# Excel常用公式速查

## 数据清洗

| 需求 | 公式/方法 |
|------|----------|
| 去除空格 | =TRIM(A1) |
| 文本转数值 | =VALUE(A1) 或 *1 |
| 合并单元格 | =A1&B1 或 =CONCATENATE(A1,B1) |
| 拆分文本 | 数据 → 分列 → 分隔符 |
| 去重 | 数据 → 删除重复项 |

## 数据分析

| 需求 | 公式 |
|------|------|
| 求和 | =SUM(A1:A100) |
| 条件求和 | =SUMIF(A:A,">=60",B:B) |
| 计数 | =COUNT(A1:A100) |
| 条件计数 | =COUNTIF(A:A,"已完成") |
| 平均值 | =AVERAGE(A1:A100) |
| 最大/最小 | =MAX(A:A) / =MIN(A:A) |

## VLOOKUP 用法

```excel
=VLOOKUP(查找值, 表格区域, 列号, FALSE)
```

**示例**：
```excel
=VLOOKUP(D2, A:B, 2, FALSE)
```
在A:B区域查找D2的值，返回第2列

**注意**：
- 查找区域第一列必须是查找值所在的列
- 第三个参数是返回第几列，不是列字母

## INDEX+MATCH（更灵活）

```excel
=INDEX(返回列, MATCH(查找值, 查找列, 0))
```

**示例**：
```excel
=INDEX(B:B, MATCH(D2, A:A, 0))
```

## IF 嵌套

```excel
=IF(条件1, 结果1, IF(条件2, 结果2, 结果3))
```

**评分示例**：
```excel
=IF(A1>=90,"A",IF(A1>=80,"B",IF(A1>=60,"C","D")))
```

## 日期处理

| 需求 | 公式 |
|------|------|
| 今天是几号 | =TODAY() |
| 现在时间 | =NOW() |
| 计算天数差 | =DATEDIF(A1,B1,"D") |
| 提取年月 | =TEXT(A1,"YYYYMM") |
