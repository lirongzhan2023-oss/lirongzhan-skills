# Excel数据技能

## 核心能力

### 1. 数据清洗

- **去重**：删除重复项、标记重复
- **填充**：批量填充空值、向下/向上填充
- **拆分**：分列功能、按分隔符拆分
- **合并**：Concatenate或&符号合并
- **格式转换**：文本转数字、日期格式统一

### 2. 数据分析

- **基础统计**：SUM、AVERAGE、COUNT、MAX、MIN
- **条件统计**：SUMIF、COUNTIF、AVERAGEIF
- **筛选排序**：多条件筛选、自定义排序
- **条件格式**：高亮重复值、数据条、色阶
- **数据透视表**：快速汇总、分组、计算

### 3. 公式函数

- **查找引用**：VLOOKUP、HLOOKUP、INDEX+MATCH、XLOOKUP
- **逻辑函数**：IF、AND、OR、IFS、NESTED IF
- **文本函数**：LEFT、RIGHT、MID、LEN、TRIM、TEXT
- **日期函数**：TODAY、NOW、DATEDIF、DATE
- **数组公式**：FILTER、SORT、UNIQUE（新版Excel）

### 4. 图表制作

- **基础图表**：柱状图、折线图、饼图、散点图
- **进阶图表**：组合图、漏斗图、帕累托图
- **图表美化**：配色、标签、趋势线
- **动态图表**：切片器、图表联动

### 5. 效率提升

- **快捷键**：Ctrl+C/V、Ctrl+E智能填充、F4绝对引用
- **格式刷**：快速复制格式
- **模板**：建立自己的常用模板
- **打印**：打印区域设置、页眉页脚
