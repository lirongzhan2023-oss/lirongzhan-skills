# 客户服务技能

## 核心能力

### 1. 接待服务

**服务流程**：

| 步骤 | 动作 | 话术 |
|------|------|------|
| 1. 迎接 | 微笑点头 | "欢迎光临，请问几位？" |
| 2. 引座 | 手势指引 | "请跟我来" |
| 3. 递水 | 倒水递巾 | "先喝口水" |
| 4. 点餐 | 递菜单 | "这是菜单" |
| 5. 推荐 | 介绍菜品 | "我们招牌是..." |
| 6. 下单 | 确认订单 | "您确认一下" |
| 7. 上菜 | 报菜名 | "您点的XX" |
| 8. 巡台 | 加水收盘 | "需要加点什么？" |
| 9. 结账 | 确认金额 | "一共XX元" |
| 10. 送客 | 感谢 | "欢迎下次再来" |

**服务细节**：
- 15分钟内上第一道菜
- 菜品上齐后询问"是否需要加菜"
- 客人离桌后30秒内收拾

### 2. 好评引导

**好评话术**：

| 场景 | 话术 |
|------|------|
| 用餐结束 | "吃得怎么样？满意的话帮忙点个好评~" |
| 上菜快 | "今天上菜挺快的，给个好评鼓励一下~" |
| 客人开心 | "看您吃得很开心，给个好评支持下~" |
| 打包时 | "打包回去也欢迎来点评~" |

**好评激励**：

| 方式 | 内容 |
|------|------|
| 口头感谢 | "谢谢您！" |
| 小礼物 | 糖果、优惠券 |
| 积分 | 会员积分奖励 |

**差评预防**：
- 上菜前确认口味
- 巡台时主动询问
- 结账时再次确认

### 3. 投诉处理

**处理流程**：
```
1. 倾听 → 不打断，让客户说完
2. 道歉 → 真诚道歉，不推卸
3. 理解 → 表示理解客户感受
4. 解决 → 提出解决方案
5. 跟进 → 确认客户满意
6. 记录 → 记录问题，避免再犯
```

**常见问题解决方案**：

| 问题 | 解决方案 |
|------|----------|
| 等太久 | 立即催厨房+免单/送菜 |
| 菜凉了 | 重新做一份+道歉 |
| 味道不对 | 重新做一份+道歉 |
| 菜里有异物 | 重新做+免单+道歉 |
| 上错菜 | 重新上+道歉+送饮料 |
| 服务态度差 | 换人服务+道歉 |

**补偿标准**：

| 严重程度 | 补偿 |
|----------|------|
| 一般 | 送饮料/小菜 |
| 较重 | 免单/重做 |
| 严重 | 免单+优惠券+道歉 |

### 4. 客户维护

**回访机制**：

| 客户类型 | 回访方式 | 时机 |
|----------|----------|------|
| 新客 | 短信 | 消费后1天 |
| 老客 | 微信 | 节日/生日 |
| 投诉客 | 电话 | 处理后1天 |

**短信模板**：
```
"尊敬的XX，感谢您XX日光临！您的意见对我们很重要，欢迎随时反馈。期待下次为您服务！"
```

**生日营销**：
```
生日前3天推送：
"亲爱的XX，生日快乐！本月到店消费享8折优惠，祝您生日月开心！"
```

### 5. 外卖客服

**好评回复模板**：

| 评价 | 回复 |
|------|------|
| 好吃 | "感谢您的认可！您的喜爱是我们最大的动力~" |
| 包装好 | "谢谢！包装我们一直很用心~" |
| 速度快 | "出餐快是因为提前备好食材，绝不将就！" |
| 复购 | "欢迎下次再来~" |

**差评回复模板**：

| 问题 | 回复 |
|------|------|
| 少送 | "非常抱歉漏送，已安排退款，下次点单联系客服补发" |
| 味道一般 | "抱歉口味没达到预期，我们会继续改进" |
| 配送慢 | "抱歉让您在高峰期等待，已反馈配送优化" |
| 撒了 | "非常抱歉，已安排退款，欢迎下次点单全额退款" |

**回复原则**：
- 30分钟内回复
- 态度真诚
- 解决问题
- 不删评
