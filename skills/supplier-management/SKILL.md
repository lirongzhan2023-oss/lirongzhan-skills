# 供应商管理技能

## 1. 供应商筛选

### 评估维度

| 维度 | 权重 | 检查项 |
|------|------|--------|
| 价格 | 40% | 同类最低、报价透明 |
| 质量 | 30% | 样品合格、稳定 |
| 配送 | 20% | 准时、响应快 |
| 服务 | 10% | 配合度、售后 |

### 筛选流程

```
1. 收集3-5家供应商信息
2. 要求提供样品
3. 对比报价
4. 试用1-2周
5. 确定合作
```

---

## 2. 议价技巧

### 常用策略

| 策略 | 说明 | 效果 |
|------|------|------|
| 多家比价 | 让供应商知道你在比价 | -10% |
| 批量采购 | 承诺稳定订单 | -15% |
| 长期合作 | 签订年度合同 | -10% |
| 现金付款 | 缩短回款周期 | -5% |
| 竞标 | 让供应商竞争 | -20% |

### 话术

```
"我们月采购量XXX，正在找长期合作伙伴，你们能做多少？"
"XX供应商给的价格是XX，你们能更低吗？"
"如果我们签一年合同，能给什么价？"
```

## 3. 供应商评估

### 月度评估表

| 供应商 | 价格 | 质量 | 配送 | 服务 | 综合 | 状态 |
|--------|------|------|------|------|------|------|
| A | 90 | 95 | 90 | 85 | 90 | ✅ |
| B | 85 | 80 | 75 | 80 | 80 | ⚠️ |
| C | 70 | 90 | 60 | 70 | 73 | ❌ |

### 评估标准

| 分数 | 等级 | 动作 |
|------|------|------|
| 90+ | 优秀 | 主力供应商 |
| 70-89 | 合格 | 保持合作 |
| 60-69 | 警告 | 限期整改 |
| <60 | 淘汰 | 更换供应商 |

## 4. 合同管理

### 必含条款

| 条款 | 说明 |
|------|------|
| 价格 | 单价、是否含税 |
| 送货 | 时间、地点、验收标准 |
| 付款 | 周期、方式、账期 |
| 违约 | 延迟、质量问题处理 |
| 终止 | 提前通知期 |

### 报价单模板

```
【供应商报价单】
- 名称：
- 联系人：
- 商品名称 | 规格 | 单价 | 备注
- 虾仁 | 1kg/包 | XX元 | 
- 芝士 | 1kg/袋 | XX元 |

有效期：XXX
付款周期：XXX
```

## 5. 备选方案

### 储备原则

```
每类至少2家供应商：
- 主力供应商（70%采购量）
- 备选供应商（30%采购量）
- 应急供应商（临时用）
```

### 应急处理

| 情况 | 处理 |
|------|------|
| 主力缺货 | 立即启用备选 |
| 质量有问题 | 采样留证，投诉 |
| 突然涨价 | 启用备选谈判 |
