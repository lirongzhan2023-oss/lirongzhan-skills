# 团队管理技能

## 核心能力

### 1. 招聘面试

**岗位描述模板**：

| 岗位 | 职责 | 要求 | 薪资 |
|------|------|------|------|
| 后厨 | 备餐、出餐、卫生 | 经验优先 | 4-6K |
| 前厅 | 点餐、服务、清洁 | 热情大方 | 3-5K |
| 店长 | 管理、运营、营销 | 1年以上 | 6-10K |

**面试问题**：

| 类型 | 问题 | 目的 |
|------|------|------|
| 经验 | "上一份工作主要做什么？" | 了解经验 |
| 动机 | "为什么想来做餐饮？" | 了解意愿 |
| 态度 | "你能接受加班吗？" | 了解预期 |
| 团队 | "和同事有矛盾怎么处理？" | 了解性格 |
| 场景 | "如果客人投诉会怎么做？" | 了解应变 |

**面试技巧**：
- 看态度 > 经验
- 试工期：3天试岗期
- 谈清楚：工资、工作时间、休息

### 2. 员工培训

**新员工入职培训**：

| 天数 | 内容 | 负责人 |
|------|------|--------|
| 第1天 | 熟悉环境、认识同事 | 店长 |
| 第2天 | 规章制度、仪容仪表 | 店长 |
| 第3天 | 岗位流程、实操练习 | 师傅 |
| 第4-7天 | 独立操作、师傅带 | 师傅 |
| 第2周 | 考核、转正 | 店长 |

**培训内容**：

**前厅培训**：
- 接待流程
- 点餐话术
- 上菜顺序
- 结账流程
- 突发情况处理

**后厨培训**：
- 食品安全
- 设备使用
- 标准化流程
- 卫生要求

### 3. 排班管理

**排班原则**：
```
1. 营业高峰：多排人
2. 营业低谷：少排人
3. 员工需求：尽量满足
4. 公平原则：轮流排班
```

**排班模板**：

| 岗位 | 早班 | 中班 | 晚班 |
|------|------|------|------|
| 前厅 | 9-17点 | 11-14,17-21 | 16-22点 |
| 后厨 | 9-14点 | 10-14,16-20 | 16-22点 |

**考勤制度**：
- 迟到：10分钟内警告
- 迟到30分钟以上：扣款
- 请假：提前1天申请
- 旷工：当天扣3天工资

---

### 4. 绩效管理

**考核指标**：

| 岗位 | 考核项 | 权重 |
|------|--------|------|
| 前厅 | 销售额、好评率、仪容仪表 | 各33% |
| 后厨 | 出餐速度、餐品质量、卫生 | 各33% |
| 店长 | 销售额、利润率、员工稳定性 | 各33% |

**激励方案**：

| 激励 | 条件 | 金额/内容 |
|------|------|-----------|
| 全勤奖 | 满勤 | 200元 |
| 销售冠军 | 月销售最高 | 300元 |
| 好评奖 | 好评≥10/月 | 100元 |
| 进步奖 | 考核提升 | 200元 |

**晋升通道**：
```
服务员 → 组长 → 店长
厨工 → 厨师 → 厨师长
```

---

### 5. 离职处理

**离职流程**：
```
1. 员工提出（书面）
2. 店长面谈（了解原因）
3. 工作交接（1周）
4. 结算工资
5. 退群、物品归还
```

**面谈技巧**：
```
"感谢你为公司付出/
想了解一下为什么离职？/
对公司有什么建议？/
随时欢迎回来。"
```

**挽留策略**：
- 了解真实原因
- 针对原因解决
- 给出发展机会
